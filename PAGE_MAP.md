# 페이지 맵 & 유저 플로우

## 📍 페이지 맵 (Page Map)

### 현재 구현된 페이지

| 페이지명 | URL | 타입 | 설명 | 상태 |
|---------|-----|------|------|------|
| 홈 | `/` | Public | 블로그 소개 및 최근 글 | ✅ 완료 |
| 글 목록 | `/posts` | Public | 모든 글 조회 (검색 기능 포함) | ✅ 완료 |
| 글 상세 | `/posts/[id]` | Public | 특정 글 읽기 | ✅ 완료 |
| 글 작성 | `/posts/new` | Protected | 새 글 작성 폼 | ✅ 완료 |
| 글 수정 | `/posts/[id]/edit` | Protected | 글 수정 폼 | ✅ 완료 |

### 구현 필요한 페이지

| 페이지명 | URL | 타입 | 설명 | 우선순위 |
|---------|-----|------|------|---------|
| 로그인 | `/login` | Public | 이메일/비밀번호 로그인 | 🔴 높음 |
| 회원가입 | `/signup` | Public | 새 계정 생성 | 🔴 높음 |
| 마이페이지 | `/profile` | Protected | 내 정보, 내 글 목록 | 🔴 높음 |

### 권장 추가 페이지

| 페이지명 | URL | 타입 | 설명 | 이유 |
|---------|-----|------|------|------|
| 사용자 프로필 | `/users/[id]` | Public | 특정 사용자의 글 목록 | 타 사용자 글 확인 가능 |
| 404 에러 | `/404` | Error | 페이지 없음 | 예외 처리 |

## 👥 주요 유저 플로우

### 1️⃣ 글 읽기 (비로그인 사용자)

```
홈 (/)
  ↓
글 목록 (/posts)
  ├─ 검색 또는 스크롤
  ↓
글 상세 (/posts/[id])
  ├─ 글 읽기
  ├─ 댓글 읽기 (향후)
  ├─ 작성자 프로필 (선택)
  └─ 목록으로 돌아가기
```

**접근 권한**: 모두 가능
**필요한 기능**: 없음

---

### 2️⃣ 글 작성 (로그인 필수)

```
홈 (/)
  ↓
로그인 필요 감지
  ↓
로그인 페이지 (/login)
  ├─ 로그인 또는
  └─ 회원가입 (/signup)
      ↓
글 목록 (/posts)
  ↓
[새 글 작성] 버튼
  ↓
글 작성 폼 (/posts/new)
  ├─ 제목, 내용, 작성자 입력
  ├─ 작성
  ↓
글 상세 (/posts/[id]) - 새 글
  └─ 작성 완료 확인

또는 수정:
글 상세 (/posts/[id])
  ↓
[수정] 버튼
  ↓
글 수정 폼 (/posts/[id]/edit)
  ├─ 내용 수정
  ├─ 저장
  ↓
글 상세 (/posts/[id]) - 수정 완료
```

**접근 권한**: 로그인한 사용자만
**필요한 기능**: 
- 인증 (Supabase Auth)
- 글 작성/수정 권한 확인 (본인만 수정 가능)

---

### 3️⃣ 마이페이지 (로그인 필수)

```
어디서나
  ↓
헤더의 [마이페이지] 또는 프로필 메뉴
  ↓
마이페이지 (/profile)
  ├─ 내 정보 확인
  │   ├─ 이름, 이메일
  │   ├─ 가입일
  │   └─ 작성한 글 수
  │
  ├─ 내가 작성한 글 목록
  │   ├─ 제목, 작성일
  │   ├─ 수정 버튼
  │   └─ 삭제 버튼
  │
  ├─ 내 정보 수정 (향후)
  └─ 로그아웃
```

**접근 권한**: 로그인한 사용자만 (본인 정보만 조회 가능)
**필요한 기능**:
- 인증 확인
- 사용자 정보 조회
- 내 글 목록 조회

---

## 🔐 접근 권한 정리

### Public 페이지 (누구나 접근 가능)

- `/` - 홈
- `/posts` - 글 목록
- `/posts/[id]` - 글 상세
- `/login` - 로그인
- `/signup` - 회원가입
- `/users/[id]` - 사용자 프로필 (추천)

### Protected 페이지 (로그인 필수)

- `/posts/new` - 글 작성
- `/posts/[id]/edit` - 글 수정 (본인 글만)
- `/profile` - 마이페이지
